# Atalhos de teclado

<div class="nbx-keys" markdown>

## Global

| Tecla | Ação |
|-----|--------|
| `Q` | Sair da TUI atual |
| `Ctrl+C` | Sair |
| `/` | Focar a barra de pesquisa |
| `G` | Focar a árvore de navegação |
| `S` | Focar a tabela de resultados |
| `R` | Atualizar o recurso atual |
| `Ctrl+G` | Limpar o log ou área de saída visível quando a visão suporta |

## Árvore de navegação

| Tecla | Ação |
|-----|--------|
| `↑` / `↓` | Mover entre nós |
| `←` / `→` | Recolher / expandir grupo |
| `Enter` | Selecionar recurso (carregar resultados) |

## Tabela de resultados

| Tecla | Ação |
|-----|--------|
| `↑` / `↓` | Mover entre linhas |
| `Space` | Alternar seleção da linha |
| `A` | Alternar selecionar todas as linhas visíveis |
| `D` | Ir para a aba Detalhes |
| `Enter` | Abrir visão de detalhe da linha selecionada |
| `F` | Abrir modal de filtros |

## Aba Detalhes

| Tecla | Ação |
|-----|--------|
| `↑` / `↓` | Rolagem dos atributos |
| `Escape` | Voltar à aba Resultados |

## Modal de filtros

| Tecla | Ação |
|-----|--------|
| `Enter` | Aplicar filtro |
| `Escape` | Fechar sem aplicar |

## Seletor de tema

| Tecla | Ação |
|-----|--------|
| `Enter` (no menu Theme) | Abrir lista de temas |
| `↑` / `↓` | Navegar temas |
| `Enter` | Aplicar tema selecionado |

</div>

---

!!! note
    Os atalhos aqui descrevem a TUI do navegador principal. Aplicações especializadas como
    o visualizador de logs, o construtor de CLI e o navegador de modelos Django mantêm as mesmas convenções
    básicas de navegação Textual, mas expõem ações diferentes específicas da visão.
