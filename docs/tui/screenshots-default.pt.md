# Capturas de tela: TUI padrão

A TUI padrão é a interface principal de navegação por recursos NetBox. Espelha o layout da UI web do NetBox com uma árvore de navegação à esquerda e um workspace com abas no centro. A TUI descobre recursos de plugins dinamicamente e pode navegar qualquer API REST sob `/api/plugins/`.

## Comando de lançamento

```bash
nbx tui                    # perfil default
nbx demo tui               # perfil demo (demo.netbox.dev)
nbx tui --theme dracula    # com tema específico
```
