# Capturas de tela: TUI GraphQL

A TUI GraphQL é um explorador GraphQL interativo para o NetBox. Combina navegação de esquema ao vivo,
montagem guiada de consultas, edição de variáveis e visualização formatada da resposta
em uma interface de terminal em tela cheia dedicada.

## Comando de lançamento

```bash
nbx graphql tui              # perfil default
nbx demo graphql tui         # perfil demo (demo.netbox.dev)
nbx graphql tui --theme dracula
```

## O que as capturas mostram

O harness de captura inicializa a TUI GraphQL com payload de introspecção determinístico
e consulta de exemplo para a galeria permanecer estável entre capturas de tema
representando ainda o layout real da TUI GraphQL.
