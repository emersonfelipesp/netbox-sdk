# Capturas de tela: Dev TUI

A Dev TUI é a bancada de requisições do desenvolvedor para exploração da API. Oferece interface interativa para testar e depurar requisições à API NetBox, examinar respostas e explorar o esquema OpenAPI. É particularmente útil para entender como a API funciona e para desenvolver integrações.

## Comando de lançamento

```bash
nbx dev tui              # perfil default
nbx demo dev tui        # perfil demo (demo.netbox.dev)
nbx dev tui --theme dracula
```
