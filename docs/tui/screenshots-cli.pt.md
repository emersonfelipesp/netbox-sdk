# Capturas de tela: Construtor de CLI

O construtor de CLI é a interface guiada orientada a menu para montar e executar
comandos `nbx` de forma interativa.

## Comando de lançamento

```bash
nbx cli tui
nbx cli tui --theme dracula
nbx demo cli tui
```
