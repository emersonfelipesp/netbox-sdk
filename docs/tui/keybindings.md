# Keyboard Shortcuts

<div class="nbx-keys" markdown>

## Global

| Key | Action |
|-----|--------|
| `Q` | Quit the current TUI |
| `Ctrl+C` | Quit |
| `/` | Focus the search bar |
| `G` | Focus the navigation tree |
| `S` | Focus the results table |
| `R` | Refresh current resource |
| `Ctrl+G` | Clear the visible log or output area when that view supports it |

## Results table

| Key | Action |
|-----|--------|
| `в†‘` / `в†“` | Move between rows |
| `Space` | Toggle row selection |
| `A` | Toggle select all visible rows |
| `D` | Jump to the Details tab |
| `Enter` | Open detail view for selected row |
| `F` | Open filter modal |
