# Keyboard Shortcuts

<div class="nbx-keys" markdown>

## Global

| Key | Action |
|-----|--------|
| `Q` | Quit the TUI |
| `Ctrl+C` | Quit |
| `/` | Focus the search bar |
| `G` | Focus the navigation tree |
| `S` | Focus the results table |
| `R` | Refresh current resource |
| `Ctrl+G` | Clear the log / output area |

| Key | Action |
