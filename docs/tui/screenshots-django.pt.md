# Capturas de tela: Navegador de modelos Django

O navegador de modelos Django é a TUI voltada a contribuidores para inspecionar
o grafo interno de modelos Django do NetBox, relacionamentos e trechos de fonte.

## Comando de lançamento

```bash
nbx dev django-model tui
```
